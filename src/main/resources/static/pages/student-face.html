🔒 รูปภาพของคุณจะถูกเข้ารหัสและเก็บใน AWS S3 (กทม.) ใช้เพื่อยืนยันตัวตนในการเช็คชื่อเท่านั้น อ่านนโยบาย PDPA เพิ่มเติม
กำลังอัพโหลด...
🔒
นโยบายความเป็นส่วนตัว (PDPA)
พระราชบัญญัติคุ้มครองข้อมูลส่วนบุคคล พ.ศ. 2562
👤 ข้อมูลที่เราเก็บ
ภาพถ่ายใบหน้าของคุณ และ Feature Vector (ข้อมูลเชิงตัวเลขที่สกัดจากใบหน้า) ที่สร้างโดย AWS Rekognition ซึ่งถือเป็น ข้อมูลชีวมิติ (Biometric Data) ตาม PDPA
🎯 วัตถุประสงค์การใช้งาน
ใช้เพื่อยืนยันตัวตนในระบบเช็คชื่อเรียน (AttendX) เท่านั้น ไม่นำไปใช้เพื่อวัตถุประสงค์อื่นใดโดยไม่ได้รับความยินยอม
🗄 การจัดเก็บข้อมูล
รูปภาพถูกเข้ารหัส ข้อมูลจะถูกลบโดยอัตโนมัติเมื่อสิ้นสุดภาคการศึกษา หรือเมื่อคุณร้องขอ
🔐 การเข้าถึงข้อมูล
เฉพาะระบบ AttendX และผู้ดูแลระบบที่ได้รับอนุญาตเท่านั้น ที่สามารถเข้าถึงข้อมูลของคุณได้ ไม่มีการเปิดเผยต่อบุคคลที่สามโดยไม่ได้รับอนุญาต
✋ สิทธิ์ของคุณ
คุณมีสิทธิ์เข้าถึง แก้ไข ลบ หรือขอสำเนาข้อมูลของคุณได้ตลอดเวลาผ่านหน้านี้ หรือติดต่อผู้ดูแลระบบ
หากไม่ยินยอม คุณจะไม่สามารถใช้ระบบเช็คชื่ออัตโนมัติได้
🗑
ยืนยันการลบรูป
